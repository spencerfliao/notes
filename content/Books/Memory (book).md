### Chapter 6 The Seven Ages Of Man

Short-term memory seems to remain quite well preserved in older individuals,

Performance on explicit long-term memory (i.e. memory with awareness of the memory experience; see Chapter 2) tasks typically declines significantly, especially on measures of free recall, although recognition holds up well with age.

Implicit memory (i.e. memory without awareness, typically tested indirectly via the evaluation of changes in behaviour rather than recollection of the memory experience) seems to decline little with age.

There is little effect of ageing on semantic memory. In fact, this capacity seems to improve throughout life.

In addition, mental exercise (such as crosswords, chess–and learning news skills such as information technology) may well be useful in maintaining neurological and psychological capacity.
